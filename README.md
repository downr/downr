
# Downr

> Fast, privacy-focused YouTube video downloader that runs in your browser

[![Website](https://img.shields.io/badge/Visit-downr.org-blue?style=for-the-badge)](https://downr.org)

## 🌐 Official Website

**[downr.org](https://downr.org)**

## ✨ Features

- 🚀 **Browser-based** - No software installation needed
- 🎥 **HD/4K/8K support** - Download in highest quality available (Currently Browser extension only feature for YouTube)
- 🔒 **Privacy-first** - No tracking or data collection
- 🚫 **Ad-free** - Clean interface, zero ads or watermarks
- ⚡ **Fast downloads** - Direct connections, no proxy delays
- 🌍 **Cross-platform** - Works on any device with a browser

## 🎯 How to Use

1. Visit **[downr.org](https://downr.org)**
2. Paste your YouTube video URL
3. Select quality (144p to 8K)
4. Click download - done!

## 💡 Why Downr?

Unlike desktop YouTube downloaders that require installation and often bundle malware, Downr runs entirely in your browser. No permissions needed, no security risks, no bloated software.

## 📋 Use Cases

- Content creators archiving their work
- Researchers building offline video libraries
- Educators creating teaching materials
- Travelers downloading content for offline viewing
- Anyone who values privacy and simplicity
